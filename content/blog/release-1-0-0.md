We're proud to announce the first public release of **Linux-Priv 1.0.0 "Zero"**.

Linux-Priv is a privacy-focused, ethical hacking distribution based on Linux Mint 22.3 Cinnamon. It ships with a hardened kernel, pre-configured anonymity tools, and a full suite of security and penetration testing software — all in a familiar, daily-driveable desktop.

## What's included

- Tor Browser, ProxyChains, Wireguard pre-configured
- Nmap, Metasploit, Wireshark, Burp Suite CE, Aircrack-ng
- VeraCrypt, KeePassXC, GnuPG
- AppArmor profiles enabled by default
- All telemetry and tracking services removed
- LUKS2 full disk encryption at install time
